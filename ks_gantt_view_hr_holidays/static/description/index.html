Key Highlights

Action Bar

Leave Management Was Never So Easy!

  • Duration

    • Duration feature displays the number of leave day/days in the chart.

  • Grid

    • Utilizing the Grid feature, once can remove the left pane that holds columns of information about the leaves, such as the leave title, duration, start date, etc. The graphical representation of Gantt View (in the right pane) takes up the entire screen providing a focused visualization of the leaves.

  • Resource Panel

    • The Resource Panel helps in managing the leaves efficiently while checking out the working hours, leaves, leave applicants, resolving leave related bottlenecks, etc.

  • Zoom To Fit

    • Zoom To Fit option to fit the Gantt View to the screen as per the convenience of the different users.

  • Critical Path

    • Critical Path displays the leave dependencies.

    If you click on the Today button, a today-marker will highlight the graphical section in the Gantt view that reflects the current day.

  • Time Sorting

    • Time Sorting allows the users to sort the Gantt View leaves in several time patterns such as day, week, month, quarter, etc. There is a Today-marker that highlights the graphical section in the Gantt view and reflects the current day.

  • Sorting

    • Using Sorting, users can sort the Gantt table (in the left pane) in many ways like Ascending/Descending order(based on Leave Names or Leave Duration) & Old to New or New To Old(as per leave creation dates).

  • Zoom In

    • Using the Zoom In feature users can zoom into the Gantt View that takes up the full screen. In this mode, one can easily utilize the scroll bars to better visualize, manage and present leaves.

Gantt View

Modify The View As Per Your Wish!

  • Create

    • The Create button helps to create a leave record as you fill the required fields in the leave form.

  • Export

    • The Gantt View Time Off report can be exported in various formats like Excel, PDF, PNG, JSON, MS Project(XML), Primavera P6, and iCal.

  • Leave Tooltip

    • The Hover Tooltip feature displays the leave details such as Start Date & the End Date of the leave as the mouse is moved over a leave-bar in the Gantt View . There is also an edit & delete option to modify or delete the project.

  • Leave Color

    • Based on the color assigned to different states, the leave bar color in Gantt View will automatically change.

  • Keyboard Shortcuts

    • There are several Keyboard shortcuts that can be used to perform the tasks that one has to perform manually using the functionalities of Gantt View.

Gantt Settings

Customize The Gantt View

  • Tooltip Settings

    • The Tooltip Settings allows you to customize the leave information display in the Tooltip view by enabling/disabling the information. If you turn on these options, the related information will be shown in the Tooltip view.

  • Dynamic Text

    • The leave description is often too long and does not fit the leave bar in the Gantt View. If you turn on the Dynamic Text, the name will be adjusted and displayed beside it or else it will be shown within the bar.

  • Quick Info

    • Quick Info feature to quickly see the information of a leave in the Gantt View(Start & End date, leave type) and edit/delete options on the go.

Q.4. How to create time off request?

These are the steps to create the time off request:
1. Click on the create/plus icon button on the Gantt view,
2. Select the leave type,
3. Add leave start/end date,
4. Add description(regarding the leave) and then save it.
